## Project Context

This is a Python-based E-Paper Display (EPD) emulator for Waveshare displays. The core module is `epd_emulator/epdemulator.py` and supports both Tkinter and Flask rendering.

## Quick Reference

- **Language**: Python 3
- **Dependencies**: Pillow, Flask (see `requirements.txt`)
- **Run demo**: `python "waveshare_emulator demo.py"`
- **License**: MIT
